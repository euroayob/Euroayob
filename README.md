# AI European Job Hunter

🚀 The ultimate AI-powered platform for Moroccan job seekers to find full-time European jobs with visa sponsorship.

## Vision

Help job seekers find, apply, and secure European positions faster with:
- **AI-powered job matching** across 25+ European countries
- **Automated resume & cover letter generation** in 14 languages
- **Intelligent application tracking** with Kanban pipeline
- **Interview preparation** with AI coaching
- **Document & visa management**
- **Email & calendar integration**
- **24/7 job search automation**

## Tech Stack

### Frontend
- Next.js 14+ (React), TailwindCSS, Framer Motion, Redux

### Backend
- Node.js, Express.js, PostgreSQL, Redis, Bull Queue

### AI Services
- OpenAI GPT-4, Claude 3, Gemini

### Infrastructure
- Docker, GitHub Actions, Vercel, Railway

## Getting Started

```bash
git clone https://github.com/euroayob/Euroayob.git
cd Euroayob
docker-compose up
```

Access at `http://localhost:3000`
